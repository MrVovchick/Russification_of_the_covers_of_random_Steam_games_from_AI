# О проекте
Я взял свою библиотеку Steam и начал в случайном порядке доставать из неё игры. Для каждой ChatGPT 5.5 получал вертикальный постер, горизонтальный и логотип игры, а также задачу — русифицировать, сохранив стиль.

# Чё внутри
- Папки с готовыми русифицированные обложками и логотипом, которые можно сразу загрузить в клиент Steam.

# Как юзать
1. Откройте с помощью WinRar'а архив.
2. Выберите понравившуюся игру
3. Скопируйте папку из архива
4. Откройте Steam
5. Выберите туже игру
6. Жмите шестерёнку
7. Выберите Свойства
8. Раздел Кастомизация
9. Жмём изменить и заменяем обложки и логотип.
